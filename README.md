Summary
- An advanced, modular Lap Time Simulator (LTS) built in Python. This tool ingests real-world Formula 1 telemetry data to reconstruct track geometry, applies a forward-backward integration physics solver, and calculates the theoretical optimal lap time of a custom race car. The project bridges software engineering with vehicle dynamics, aerodynamics, and data visualization—mirroring the exact tools used by F1 performance engineers.

Technical Stack 
- Core Logic & Math
  - Python, NumPy, SciPy
- Data Ingestion
  - FastF1, Pandas
- Visualization
  - MatplotLib
- GUI
  - PyQt5
